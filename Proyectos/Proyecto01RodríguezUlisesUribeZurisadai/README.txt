### Descripción
Desarrollar el juego Solterona.

### Inconvenientes
El método para asignar las cartas a los jugadores de la forma más equitativa
posible salía demasiado largo, al momento de que el usuario ganara y la computadora
siguiera, se creaba un bucle o mandaba error.


### Compilación y ejecución
Para ejecutar esta práctica se utilizarán los siguientes comandos:
* ant build - Para crear la carpeta en donde irán los .class y .jar
* ant jar - Para compilar la práctica.
* ant run - Para ejecutar la práctica.
* ant clean - Para limpiar el proyecto.
